  v-time=''
  格式：年-月-日 时:分:秒 或 时间戳
  
  从代码中，我们可以看到，这是将代码封装成了一个指令，且加入“
  每隔一分钟更新一次”的功能，也就是说每分钟都会更新一下信息的
  发布时间距离当前时间的时间差。

  该指令会实时将信息发布的时间转换成几分钟前、几小时前、几天前，
  但是超过一个月的，就展示的是信息发布时的带有日期格式的日期了。
